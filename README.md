# 🎓 EduVerse – Online Learning Platform

EduVerse is a **full-stack MERN-based Learning Management System (LMS)** designed to provide a modern, responsive, and scalable online learning experience.

The project includes a **React frontend styled with Tailwind CSS** and a **Node.js + Express backend** connected to MongoDB.  
The frontend is deployed on **Vercel**.

## ✨ Features

- 🔐 User Authentication (Login / Register)
- 📚 Course & lesson management
- 🧠 Interactive learning flow
- 🎨 Fully responsive UI using Tailwind CSS
- 🌐 RESTful APIs
- ⚡ Fast and optimized frontend
- 🚀 Production-ready deployment

## 🛠 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- React Router DOM

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Deployment
- Frontend: **Vercel**
- Backend: Local / Render / Railway (recommended)
